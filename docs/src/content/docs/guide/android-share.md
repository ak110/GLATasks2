---
title: Android 共有
description: Android の共有メニューから GLATasks にタスクを追加する方法
---

PWA としてホーム画面に追加すると、Android の共有メニューから直接タスクを追加できる（Web Share Target API を使用）。

## セットアップ

1. Android の Chrome で GLATasks を開く
2. ブラウザのメニューから「ホーム画面に追加」を選択
3. GLATasks が PWA としてインストールされる

## 使い方

1. 共有したいコンテンツ（Web ページ、テキストなど）を開く
2. Android の共有メニューを開く
3. 「GLATasks」を選択
4. タスクとして取り込まれる
